# Belagavi Organics Website

A professional, multi-language (English, Hindi, Marathi, Kannada) organic store website for **Belagavi Organics**.

## 🚀 Deployment Instructions

To share this website with your friends and customers, follow these steps:

### Step 1: Install Git
If you don't have Git installed, download it here: [https://git-scm.com/](https://git-scm.com/)

### Step 2: Initialize & Push to GitHub
1. Create a new repository on [GitHub](https://github.com/new). Name it `belagavi-organics`.
2. Open PowerShell or Command Prompt in this folder.
3. Run the following commands:

```powershell
# 1. Initialize Git
git init

# 2. Add all files
git add .

# 3. Commit the changes
git commit -m "Initial commit: Professional Multi-language Organic Website"

# 4. Link to your GitHub (Replace YOUR_USERNAME with your real github name)
git remote add origin https://github.com/YOUR_USERNAME/belagavi-organics.git

# 5. Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Hosting (Share the link)
1. Go to your repository on GitHub.
2. Click **Settings** > **Pages**.
3. Under **Branch**, select `main` and `/ (root)`, then click **Save**.
4. Wait 1-2 minutes. GitHub will give you a link like `https://YOUR_USERNAME.github.io/belagavi-organics/`.

**You can now share this link with anyone!** 🌿✨
